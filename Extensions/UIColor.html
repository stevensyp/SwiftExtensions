UIColor

extension UIColor

Extends the UIColor class to enable new types of initializers and add a retro-compatibility to the new dark-mode compatible system colors added in iOS 13.

Initializers

  • init(hex:alpha:)

    Initialize color from hexadecimal code.

    Declaration

    Swift

    convenience init(hex: Int, alpha: Float = 1.0)

    Parameters

    hex

    Hexadecimal code (ex: 0xFFFFFF)
    alpha

    Opacity (default value is 1.0)
  • init(red:green:blue:alpha:)

    Initialize color from red/blue/green model. Color values must be between 0 and 255.
